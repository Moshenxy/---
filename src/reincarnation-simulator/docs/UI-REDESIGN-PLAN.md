# UI 优化美化方案：铸造“归墟道镜”

本方案旨在将“轮回-诸天万界”的前端UI进行一次完整的**视觉重铸**，将目前功能性的“模拟器”界面，升华为一个能让玩家沉浸于诸天万界宏大叙事的“**归墟道镜**”或“**天命罗盘**”。

方案分为四个阶段执行：**1. 风格奠基**、**2. 布局重塑**、**3. 组件精雕**、**4. 动效点睛**。

---

### **第一阶段：风格奠基 - 定义“诸天万界”的视觉语言**

此阶段的目标是建立一套统一、高级且符合主题的视觉规范。

1.  **色彩体系：【玄黑为底，星辉点金】**
    *   **主背景色**: 采用带有微妙纹理的**玄黑**或**靛蓝** (`#0a0f1e`)，模拟深邃的宇宙虚空。
    *   **主面板色**: 使用带有半透明“磨砂玻璃”质感的**深青灰色** (`rgba(22, 33, 44, 0.85)`)。
    *   **核心点缀色**:
        *   **鎏金 (`#d4af37`)**: 用于最重要的操作按钮、面板标题、核心数据。
        *   **天青 (`#3a8f9d`)**: 用于次级信息、可交互元素、高亮状态。
        *   **赤红 (`#c74343`)**: 用于警示、伤害、负面状态。
    *   **文本色**: 基础文本采用**月白** (`#e0e0e0`)，重要描述使用**淡金** (`#f0e6d2`)。

2.  **字体方案：【楷书风骨，篆书点睛】**
    *   **主字体**: 选用一款清晰易读、兼具古典风韵的开源楷体或宋体。
    *   **标题/艺术字体**: 选用一款风格化的篆书或书法字体，用于各大面板的标题等关键位置。

3.  **图标设计：【万物有形，皆可为符】**
    *   设计一套全新的、基于“符文”、“星宿”、“法宝”概念的单色SVG图标库。

4.  **目录结构 (`/styles`)**:
    *   新建 `src/reincarnation-simulator/styles/theme/` 目录。
    *   `_variables.scss`: 存放所有颜色、字体、间距等设计变量。
    *   `_mixins.scss`: 存放可复用的样式片段（如磨砂玻璃效果）。
    *   `_base.scss`: 定义全局基础样式。

---

### **第二阶段：布局重塑 - 打造沉浸式“仪表盘”**

1.  **根组件 (`app.vue`)**: 应用带有星云、星图的动态背景。
2.  **侧边导航 (`SidePanel.vue`)**: 按钮重设计为“**星宿图腾**”样式（图标在上，文字在下），悬浮时有辉光效果。
3.  **主内容区 (`MainView.vue`)**: 叙事文本区域背景更通透，实体高亮效果更华丽。

---

### **第三阶段：组件精雕 - 万千法门，各有洞天**

1.  **面板通用样式**: 标题栏使用艺术字体，边框使用 `border-image` 添加流动光效。
2.  **核心面板重铸**:
    *   **角色面板 (`CharacterPanel.vue`)**: 以“**命盘**”或“**星盘**”的形式呈现“精、气、神、运”四大基础潜力。
    *   **轮回烙印 (`ImprintsPanel.vue`)**: 升级为“**九宫格**”或“**周天星斗阵**”布局，根据能级散发不同光芒。
    *   **背包面板 (`InventoryPanel.vue`)**: 物品图标根据能级拥有不同颜色的背景辉光，详情浮窗模拟古籍或玉简样式。
    *   **轮回面板 (`ReincarnationPanel.vue`)**: “世界选择”重做为可交互的“**诸天星图**”，每个世界都是一颗可交互的星辰。

---

### **第四阶段：动效点睛 - 为界面注入“灵气”**

1.  **微交互动画**: 所有可交互元素在悬浮和点击时，都有柔和的辉光、缩放或能量粒子效果。
2.  **转场与加载**: 面板打开和关闭使用淡入淡出或粒子消散效果，数据加载时显示“星云旋转”或“符文流转”的动态指示器。
3.  **通知系统 (`toastr`)**: 重写样式，使其显示为一道“**金色法旨**”或“**虚空传音**”。
