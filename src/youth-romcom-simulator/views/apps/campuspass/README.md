# 校园通 - 课程表模块

本模块负责展示学生的个人课程表。

## 文件结构

-   **`ScheduleView.vue`**: 课程表的主视图。它负责：
    -   从全局 store 中获取玩家的班级信息 (`playerClass`) 和世界状态中的课程表原始文本数据 (`课程表-主`)。
    -   实现了一个健壮的解析器 (`allCourses`)，用于将复杂的、多格式的课程表文本解析为结构化的 `Course[]` 数组。
    -   按星期 (`周一` 到 `周日`) 对课程进行分组。
    -   循环渲染每一天的课程列表，并使用 `ScheduleCard` 组件来展示单门课程。

-   **`ScheduleCard.vue`**: (新组件) 用于显示单门课程信息的卡片组件。它负责：
    -   接收一个 `course` 对象作为 prop。
    -   根据课程的节次 (`节次`) 计算并显示课程的开始和结束时间。
    -   展示课程的名称、教师和教室ID。
    -   根据课程名称的哈希值动态生成卡片的背景和边框颜色，使界面更加生动。

-   **`GradesView.vue`**: 成绩查询视图 (暂未实现)。

-   **`NotificationsView.vue`**: 校园通知视图 (暂未实现)。

## 核心重构逻辑 (V2)

-   **数据驱动**: 课程表的显示完全由 `课程表-主` 世界书条目的内容驱动。
-   **健壮的解析**: 解决了旧版本无法解析多段上课时间以及复杂格式的问题，确保所有课程都能被正确解析和显示。
-   **组件化**: 将单门课程的显示逻辑封装到独立的 `ScheduleCard.vue` 组件中，提高了代码的可读性、可维护性和复用性。
-   **现代化UI**: 放弃了旧的、复杂的日历网格布局，改为更现代、更直观的卡片式列表布局，提升了用户体验。
